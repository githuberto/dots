Dots
====

Installation
------------
First, run the scripts to install all the packages and config files.
```
./packages.sh  
./install.sh -i
```

Now run `lxappearance` and pick a theme (Adwaita-dark is nice).

Uninstallation
--------------
Run the uninstallation script to remove all the config files.
```
./install.sh -u
```
